---
title: "Portals"
description: "Portal CRUD, genie assignment, and distribution actions"
---

Manage public/private portals with the `portals` resource.

Supported actions: `all`, `list`, `get`, `create`, `update`, `delete`, `reorder`, `add-genie`, `remove-genie`, `reorder-genies`, `share-url`, `embed-code`.
