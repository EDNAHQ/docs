---
title: "Billing"
description: "Subscription, usage, and Stripe delegated billing actions"
---

Use the `billing` resource for plan retrieval and billing lifecycle actions.

Supported actions: `all`, `plan`, `usage`, `invoices`, `checkout`, `billing-portal`, `update-subscription`, `sync`, `purchase-calls`, `cancel-calls`, `add-phone-number`, `remove-phone-number`.
